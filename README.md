# Introduction

This project aims to demo how Bag-of-Words Model in NLP using nltk library including some common steps:
- Vocabulary building
- Preprocessing: remove stopwords, stemming
- Vectorization
  
## Tech stack

Python 3.8, nltk, pip

## How to get started?

Start an example of bow
```shell
pip install -r requirements.txt
python bow_app.py
```
To run unit test
```shell
pip install -r requirements.txt
python bow_test.py
```
